**Петров Матвей, группа ИПР-22-1б**

## Цель работы

Найти параметры нелинейной модели  

**y = b₁ · ln(b₂ · x)**  

с использованием **метода прямого поиска** и **симплекс-метода Нелдера–Мида**,  
сравнить их эффективность и точность.

## Результаты оптимизации

### Метод прямого поиска
*(начальные значения: b₁₀ = 2.0, b₂₀ = 1.0, Δb = 0.1, точность fₙ = 0.001)*

| Параметр     | Значение      |
|--------------|---------------|
| **b₁**       | **1.996875**  |
| **b₂**       | **0.906250**  |
| **SSE**      | **0.001264**  |
| **Итераций** | **16**        |

### Симплекс-метод Нелдера–Мида
*(начальные значения: b₁₀ = 2.0, b₂₀ = 1.0, точность 0.001)*

| Параметр     | Значение      |
|--------------|---------------|
| **b₁**       | **1.985590**  |
| **b₂**       | **0.916931**  |
| **SSE**      | **0.000993**  |
| **Итераций** | **11**        |

---

## Сравнение методов

| Критерий                     | Метод прямого поиска       | Симплекс-метод Нелдера–Мида |
|------------------------------|----------------------------|-----------------------------|
| **Точность (SSE)**           | 0.001264                   | **0.000993** (лучше)   |
| **Количество итераций**      | 16                         | **11** (меньше)             |
| **Скорость сходимости**      | Медленная                  | **Быстрая**                 |
| **Чувствительность к стартовым значениям** | Высокая          | **Низкая**                  |

---

## Вывод

**Симплекс-метод Нелдера–Мида** показал **значительно лучшую точность**  
(SSE = **0.000993** против **0.001264** у метода прямого поиска)  
и потребовал **меньше итераций** (11 против 16).

Кроме того, он менее чувствителен к выбору начальных значений параметров.

**Лучшим методом для решения данной задачи является симплекс-метод Нелдера–Мида.**
