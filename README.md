| Критерий                | Прямой поиск | Симплекс-метод |

> Метод прямого поиска нашёл чуть более точное решение (SSE = 1.067059), но потребовал больше итераций (14).  
> Симплекс-метод сошёлся быстрее (всего 11 итераций), но с небольшим проигрышем в точности.
